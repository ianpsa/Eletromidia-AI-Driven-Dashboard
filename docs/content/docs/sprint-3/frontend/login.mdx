## Papel da tela

Ponto de entrada da aplicação. Ao submeter o formulário, o componente chama `onLogin()` e altera o estado local `loggedIn` para `true`. Não há autenticação real contra backend nem persistência de sessão.

![Tela de login do frontend](/sprint-3/frontend/login.png)

## Estrutura visual

Layout em duas colunas:

| Coluna   | Conteúdo                                            |
| -------- | --------------------------------------------------- |
| Esquerda | Fundo laranja, logo e card com formulário de acesso |
| Direita  | Carrossel de imagens institucionais da Eletromidia  |

Campos do formulário: `Seu e-mail` e `Senha`, com botão para alternar visibilidade da senha.

## Comportamento

- Botão `Entrar` permanece desabilitado enquanto e-mail ou senha estiverem vazios
- Submissão não chama API — apenas avança para o `Dashboard`
- Carrossel troca a imagem ativa a cada 4 segundos

## Responsividade

Em telas menores a coluna do carrossel é ocultada, mantendo o foco no formulário.
