## Papel da tela

Primeira tela exibida após o login. Centraliza a visualização analítica via embed do Looker Studio, sem lógica de filtros ou estado de gráficos no React.

![Tela Dashboard do frontend](/sprint-3/frontend/dashboard.png)

## Estrutura visual

| Elemento       | Descrição                                     |
| -------------- | --------------------------------------------- |
| Cabeçalho      | Título `Dashboard` e subtítulo de indicadores |
| Área principal | `iframe` ocupando todo o espaço disponível    |

O conteúdo analítico é renderizado inteiramente pelo Looker Studio dentro do `iframe`.

## Comportamento

- Navegação entre `Dashboard`, `Cloud` é feita pelo menu lateral
- Filtros, gráficos e interações pertencem ao Looker Studio
- O frontend não replica nem intercepta estado do relatório

## Integração

| Recurso   | Valor                                                                                  |
| --------- | -------------------------------------------------------------------------------------- |
| Embed URL | `https://lookerstudio.google.com/embed/reporting/1776f716-b7de-4268-99ef-8107f950868d` |
| Tipo      | `iframe` — sem chamadas de API próprias                                                |
