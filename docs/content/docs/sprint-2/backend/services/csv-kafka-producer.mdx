## O que é

O csv-kafka-producer é o componente **API Producer** da [arquitetura GCP](/docs/sprint-1/arquitetura): um serviço REST em Rust que lê dados de um arquivo CSV e os produz para o Kafka, servindo como fonte de dados para todo o pipeline de telemetria OOH.

---

## API REST

### Endpoint principal

```
POST /csv
```

### Endpoint de health

```
GET /health
```
