## Objetivo da Sprint

Implementar a base técnica do sistema, criando um pipeline funcional de ingestão e visualização inicial de dados conforme definido no [planejamento do projeto](/docs/sprint-1/planejamento-projeto).

## Progresso

O desenvolvimento do backend foi concluído com a implementação de dois serviços principais que formam o pipeline de dados. O csv-kafka-producer em Rust oferece alta performance no envio de mensagens para o Kafka, enquanto o bigquery-writer em Go persiste os dados no BigQuery para análise.
