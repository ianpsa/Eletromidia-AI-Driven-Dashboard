## Progresso

O desenvolvimento do backend foi concluído com quatro serviços formando o pipeline de dados: o `csv-kafka-producer` em Rust produz eventos para o Kafka; o `bigquery-writer` em Go persiste os dados no BigQuery; o `bucket-consumer` faz backup dos eventos no Cloud Storage; e o `bff-storage` expõe os arquivos via HTTP para a interface. O dashboard analítico no Looker Studio foi entregue com visualizações interativas de fluxo de pessoas segmentado por classe social, gênero e faixa etária.
